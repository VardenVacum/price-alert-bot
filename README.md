# Price Alert Bot

Бот для мониторинга цен фьючерсов Binance с уведомлениями в Telegram и WhatsApp.

## Возможности

- Автоматический мониторинг **всех USDT фьючерсных пар** на Binance
- Простой ввод: `BTC 63500` или `Sol 150,5` — USDT добавляется автоматически
- Умное распознавание: напишите `btc 63500` в любой момент — бот поймёт
- Принимает точку и запятую как разделитель дробной части
- Звуковое уведомление (MP3) чтобы телефон воспроизвёл звук даже в беззвучном режиме
- Inline-кнопки для удобной навигации
- Алерты хранятся в JSON и не теряются при перезапуске

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и кнопки меню |
| `/alert` | Создать алерт (через интерактивный ввод) |
| `/alerts` | Показать все активные алерты |
| `/delete BTCUSDT 61000` | Удалить конкретный алерт |
| `/stop` | Удалить все алерты |

Или просто напишите `BTC 63500` — бот автоматически создаст алерт.

## Установка локально

```bash
cd price_alert_bot
pip install -r requirements.txt
```

Создайте `.env` из примера:

```bash
copy .env.example .env
```

Заполните `TELEGRAM_BOT_TOKEN` (получить у @BotFather) и запустите:

```bash
python main.py
```

## Кастомный звук

Бот отправляет MP3-файл `sounds/alert.mp3` при срабатывании алерта.
Telegram воспроизводит его как голосовое сообщение — телефон издаёт звук даже в беззвучном режиме.

### Как заменить звук на свой

1. Положите любой MP3-файл в папку `sounds/` с именем `alert.mp3`
2. Перезапустите бота

Требования к файлу:
- Формат: MP3
- Рекомендуемая длительность: 1–3 секунды
- Максимальный размер: 50 МБ (ограничение Telegram)

Если файл `sounds/alert.mp3` отсутствует — бот отправляет только текстовое уведомление.

## Деплой на бесплатный хостинг

### Вариант 1: Railway (рекомендуется)

1. Зарегистрируйтесь на [railway.app](https://railway.app)
2. Подключите GitHub-репозиторий с проектом
3. Railway автоматически определит `railway.json` и соберёт Docker-образ
4. В Variables добавьте: `TELEGRAM_BOT_TOKEN` = ваш токен
5. Нажмите **Deploy**

Railway даёт $5/мес бесплатно — хватает для 24/7 работы.

### Вариант 2: Render

1. Зарегистрируйтесь на [render.com](https://render.com)
2. **New > Background Worker**
3. Подключите GitHub-репозиторий
4. Render подхватит `render.yaml`
5. В Settings > Environment добавьте: `TELEGRAM_BOT_TOKEN` = ваш токен
6. **Deploy**

Render даёт 750 часов/мес бесплатно.
